# Portfolio Website 🚀

A modern and responsive personal portfolio website showcasing my projects, technical skills, internships, certifications, and achievements.

## 🌟 Features

* Responsive modern UI
* Skills and technology section
* Projects showcase
* Internship and experience details
* Contact section
* Smooth navigation and animations

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* React.js
* Tailwind CSS

## 📂 Sections

* Hero Section
* About Me
* Skills
* Projects
* Experience
* Contact
